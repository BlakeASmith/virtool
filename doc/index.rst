Introduction
============

Virtool is a web-based application for identifying viral presence in Illumina libraries. It is intended for use in a
diagnostic environment.

Features
--------

- Real-time communication between clients and the server.
- A non-prioritizing job management system.
- Identification of infecting viruses at the species, isolate, and sequence component levels.
- A fully editable database of viral references with change-tracking.
- A sample manager with sample-wise quality information and analysis viewing.
- A complete user management system with permissions and sample access rights.
